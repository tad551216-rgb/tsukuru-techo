# つくる手帖

70歳。スマホとAIで、自分や家族のための小さな道具をつくっています。

このリポジトリは、GitHub Pagesで公開している「つくる手帖」のサイト一式です。家計簿、予定板、作業日報、日報集計、ピンボールなど、どれもブラウザで開いて使えるPWAとしてまとめています。

## このサイトで大事にしていること

- **つくる**：既製品に合わせるのではなく、自分や家族の使い方に合わせて道具を作る。
- **のこす**：できあがったものだけでなく、考えたこと、つまずいたこと、判断したことを記録する。
- **わたす**：同じように小さな道具を作りたい人へ、AIとの付き合い方や勘所を渡す。

## 主な構成

- `index.html`：トップページ
- `articles/`：動画付きの開発記・紹介記事
- `docs/`：動画・PDF資料
- `characters.html`：つくる手帖のなかまたち
- `sitemap.html`：人が読むためのサイトマップ
- `sitemap.xml`：検索エンジン向けサイトマップ

## 作った道具

- わが家の家計簿
- わが家の予定板
- 放課後等デイサービス 作業日報
- 作業日報集計
- つくる手帖ピンボール

各アプリは、端末内保存を基本にしています。利用者の入力データをサーバーへ送らない設計を重視しています。
