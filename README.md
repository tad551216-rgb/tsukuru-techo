# つくる手帖

**AIで"自分専用"の道具をつくる。**

既製品では手の届かないところを、AIを相棒に自作し、その作り方・勘所・つまずきまで正直に残していく個人サイトです。

🔗 **公開サイト**: https://tad551216-rgb.github.io/tsukuru-techo/

---

## このサイトについて

製造業向けのシステム開発に長く携わってきた作者が、セミリタイヤ後の70歳のいま、スマートフォンとAIで、自分や家族が日々使う小さなアプリをつくっています。大きな成功を狙うのではなく、「誰か一人の役に立てば、それで価値がある」を信条に、設計の判断やつまずきまで含めて、過程ごと書き残しています。

## 3つの柱

- **つくる** — 欲しい道具を、自分で。既製品で妥協しない。
- **のこす** — 作り方を、正直に。うまくいった点も、回り道も。
- **わたす** — 勘所と必要スキルを。AIに任せる部分と、人が決める部分の線引きを共有。

## 公開中のアプリ

すべて PWA（Progressive Web App）。インストール不要で、ホーム画面に追加すればアプリのように使えます。**データは利用者の端末内にのみ保存**され、外部には送信しません。

- **わが家の家計簿** — 銀行・カードのCSVを取り込み、家計の収支から確定申告の概算までを一つに。
  https://tad551216-rgb.github.io/demo-kakeibo/
- **わが家の予定板** — 家族の予定を1枚で共有。月間カレンダー・週間リスト・写真付き日記。
  https://tad551216-rgb.github.io/yotei3/
- **放課後等デイサービス 作業日報** — 作業を15分単位で報告。勤怠・時間計算の元になる記録に特化し、個人情報や療育内容は一切扱わない。
  https://tad551216-rgb.github.io/nippo/
- **作業日報集計** — 「作業日報」の出力をLINEで集め、貼り付けるだけで従業員別・期間別に自動集計。月単位の保存、取り込み漏れの確認、日報・週報・月報のExcel出力まで。
  https://tad551216-rgb.github.io/nippo-syuukei/

## 記事

- 鉄からシステム、そしてAIへ（エッセイ）
- LINEに流れる日報を、自動で集計する仕組みをつくった話（開発記）
- 家計簿アプリを、AIと作り切るまで（開発記）

## 技術メモ

- **ホスティング**: GitHub Pages
- **アプリ形式**: PWA（オフライン対応・ホーム画面追加可）
- **データ保存**: 各アプリとも利用者の端末内のみ（IndexedDB ほか）
- **編集運用**: ローカル（iPhone / Working Copy）で編集し、GitHub へプッシュして反映
- **命名規則**: リポジトリ内のファイル名は半角英数（ローマ字）で統一。表示名のみ日本語可
